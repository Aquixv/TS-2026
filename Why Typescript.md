**Typescript is a powerful Javascript framework with multiple Advantages**
**However it has the disadvantage of not being able to be ran directly by the browser and instead must be compiled**
**Typescript compiler compiles TS to JS using workarounds**
**It is more or less and easier way to write Javascript**
**TS is strongly typed, JS is dynamically typed**
**Example being, when we declare const in JS, all we do is give it name, we do not declare what type of data it stores**
**We can change the data type in JS from string to numeric, but not for TS**
**Advantages of being strongly typed are:**
**1: Faster Bug detection**
**2: Provides Extra features unavailable with JS, Advanced JS if you'd rather**
**Highly Configurable**
**Disadvantage: Needs to be compiled to JS before Execution**
